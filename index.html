关于城汇玩使用wrk工具进行接口压力测试

wrk是一个接口压力测试工具
最好的使用方法是在docker容器中使用
使用docker pull williamyeh/wrk拉取镜像

例如使用docker run –rm williamyeh/wrk -c 10 -d 5s http://192.168.101.1/get_activity_list/v1命令来进行压力测试

其中–rm是启动一次自动删除 -c是http连接数 -d是持续时间
如需要可以加上-t使用线程数，因为是在容器中是无法访问宿主机localhost的
所以需要使用netstat -nr | grep default | awk ‘{print $2}’查看ip

关于城汇玩使用pprof工具分析项目性能瓶颈的经验

pprof工具是golang本身自带的一个工具，用于分析整体项目的性能瓶颈，分析程序中哪一部分的cpu资源占用最高。

  1. 如何解读 pprof 分析结果
    1.1 CPU Profile
    CPU Profile 用于分析程序中哪些函数占用了最多的 CPU 时间。以下是一些关键指标：
    flat：当前函数占用的 CPU 时间。
    flat%：当前函数占用 CPU 时间的百分比。
    cum：当前函数及其调用的子函数占用的总 CPU 时间。
    cum%：累计 CPU 时间的百分比。

例如：
pprof) top10
Total: 1173 samples
205 17.5% 17.5% 1083 92.3% main.FindLoops
138 11.8% 29.2% 21
在这个例子中，main.FindLoops函数占用了最多的CPU时间（92.3%，说明他就是性能瓶颈）.
